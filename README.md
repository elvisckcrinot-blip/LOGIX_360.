# LOGIX_360.
LOGIX 360 : Hub SCM 4.0 pour l'industrie au Bénin et la ZLECAF. Basée sur le standard MIT, l'app fusionne IA (Random Forest) et logistique pour maximiser la profitabilité. Avec 5 interfaces (Analytique à Systèmes), elle optimise stocks, corridors et exportations. Un outil décisionnel haute performance pour transformer la Supply Chain régionale.
